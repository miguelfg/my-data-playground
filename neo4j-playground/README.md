# Neo4j Playground

This repository contains a collection of Neo4j playgrounds and examples to help you get started with graph databases using Neo4j. Whether you're new to Neo4j or looking to explore advanced features, you'll find useful resources here.

## GDS Introduction

The `gds-introduction/` directory contains notebooks and scripts that introduce the Graph Data Science (GDS) library. These resources cover basic concepts, algorithms, and practical applications of GDS in Neo4j.
